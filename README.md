# IET-OU / open-media-player-themes

Player themes for the [Open Media Player][omp].

* http://embed.open.ac.uk


## Installation

Install and test using Git and [Composer][],

```sh
    git clone https://github.com/IET-OU/open-media-player-themes
    composer install
    composer test
```
